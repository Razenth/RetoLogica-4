# RetoLogica-4
## Reto 4: Votaciones de la CONFECH
En este ejercicio implementé directamente el ejercicio, donde apenas se ejecuta, preguntará la cantidad de universidades que participan

```terminal
Ingrese la cantidad de universidades que participan: 
```

- La persona debe ingresar el número de universidades que participan

- Después le preguntará el nombre de la universidad

```terminal
Ingrese el nombre de la universidad #1:
```
- La persona deberá después ingresar uno a uno los votos que tuvo la universidad, para luego pasar a la siguiente universidad
```terminal
Ingrese voto para la universidad UDI (A para aceptar, R para rechazar, N para nulo, B para blanco). Ingrese X para finalizar:
```

- Una vez ingresado todas las universidades y sus votos finalmente le mostrará el resultado de los votos y su total 
```terminal
Número de Universidades 2
Universidad: UDI
Aceptan: 2
Rechazan: 0
Nulos: 0
Blancos: 0
---------------------------------
Universidad: UIS
Aceptan: 2
Rechazan: 0
Nulos: 0
Blancos: 0
---------------------------------
Universidades que aceptan: 2
Universidades que rechazan: 0
Universidades con empate: 0
```
